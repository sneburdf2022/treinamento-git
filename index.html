Aula 4 - 06 de Outubro

Perguntas

  • 1 - Quais os comandos para configurar usuário e email no git?/li>

    Responsta:

    Configurar o nome - git config --global user.name "Seu nome"

    Configurar o email - git config --global user.email "email@do.usuario.com"

  • 2 - Qual a diferança entre git e github

    • Responsta:

    Git é um sistema de controle de versão distribuido

    GitHub é um serviço baseado em nuvem que hospeda um sistema de controle de versão chamado Git

  • 3 - O que faz o comando commit?

    • Responsta:

    Comando commit confirma suas modificações

  • 4 - O que faz o comando push?

    • Responsta:

    Comando push é utilizado para enviar o conteúdo do repositório local para um repositório remoto
